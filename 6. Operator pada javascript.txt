Operator pada javascript

operator dalam javascript adalah sebuah simbol yang 
digunakan untuk melakukan sebuah operasi.

jenis operator pada javascript
binary (membutuhkan 2 operand)
-> operand1 operator operand 2 / 1 + 1
 1. aritmatika => +, -, *, /, %
 2. penugasan => =(penulisan pada variabel),
                 +=, -=, *=, /=, %=
 3. perbandingan => ==, !=, ===(operator identitas), !==(strict tidak sama dengan),
                    >, <, >=, <=
    yang akan menghasilkan boolean (true dan false)
 4. logika => &&, ||, ! / and, or, not
 5. string => + (bisa memiliki 2 fungsi)

ternary (membutuhkan 3 operand)
 6. kondisional => (kondisi) ? benar : salah
                   (4 % 2 == 0) ? "genap" : "ganjil" // hasilnya genap

unary (membutuhkan 1 operand)
-> operator operand / operand operator
 7. typeof => typeof(operand)
              typeof(10) // hasilnya number
